demo database for students/teachers

There is one student account. Click 'student mode' to access it and then select a teacher from the list.

To use teacher mode, select a teacher from the list.

To create a new teacher, type a name in the box and click 'new user'.

To modify the schedule. click a cell. Remember to 'save'!

Refresh the page to return to the start and see changes reflected in the data.

Alerts show the data being loaded/saved
